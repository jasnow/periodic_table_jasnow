# PeriodicTableJasnow

Provide periodic table data.

## Installation

Add this line to your application's Gemfile:

    gem 'periodic_table_jasnow'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install periodic_table_jasnow

## Usage

periodic_table_jasnow ELEMENT
